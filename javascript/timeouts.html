Timeouts/Intervals

Timeouts and intervals are a way to make Javascript runa function at a later date while running other code in the meantime. The function in the timeout/interval is asynchronous

setTimeout

Set timeout allows to you to sepcify a function to be run once after X time in milliseconds

As I said earlier this code is async so..

setInterval

Set interval runs a function every X miliseconds

You may want to know how to stop an interval once you got it started.

setInterval returns an interval id. When you pass that id to clearInterval it stops it running

Why this is cool

Several cool things you may want to do can make use of timeouts such as:

  • Games
